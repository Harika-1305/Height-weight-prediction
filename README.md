# 📏 Height vs Weight Prediction using Linear Regression

## 📌 Project Overview
This project predicts a person's weight based on their height using a Linear Regression model. It demonstrates basic machine learning concepts including data preprocessing, model training, evaluation, and visualization.

## 📊 Dataset
- Dataset contains two features:
  - `height` (independent variable)
  - `weight` (dependent variable)
- No missing values in the dataset

## ⚙️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  
- Jupyter Notebook  

## 🚀 Steps Performed
1. Imported required libraries  
2. Loaded dataset using Pandas  
3. Checked for missing values  
4. Split data into training and testing sets  
5. Trained Linear Regression model  
6. Predicted weights using test data  
7. Evaluated model using MSE and R² score  
8. Visualized actual vs predicted values  

## 📈 Model Details
- Algorithm: Linear Regression  
- Mean Squared Error (MSE): 134.94  
- R-squared Score: 0.15  

## 📊 Visualization
- Scatter plot shows actual height vs weight  
- Regression line shows predicted values
